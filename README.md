# MACEF | VACANTES

Es proyecto es una aplicación web para que macef, una empresa de administración inmobiliaria, publique sus vacantes de empleo abiertas y reciba aplicaciones a esos puestos, por los usuarios.

## *Proyecto fron end por ahora*
Por el momento, este proyecto solo se encuentra con el front desarrollado y lo mínimo necesario del backend. La idea es preparar algo sólido y que funcione de comienzo, y luego ir escalando según se necesite.

## *Estado* 
* HTML CSS y JS vanila para el front
* nodeJS y express para el back. 
* V1.0 front 90% compleo
